Project Description

This project analyzes rehabilitation data from the Center of Specialization in Physical and Intellectual Rehabilitation (CER) at the University of Vale do Itajaí, an outpatient rehabilitation facility in southern Brazil that serves the public healthcare system. The clinic provides specialized services, including physiotherapy, psychotherapy, speech therapy, and occupational therapy, across 13 cities.

The project focuses on data preprocessing, exploratory data analysis (EDA), feature engineering, and machine learning to identify patterns, generate insights, and support data-driven decision-making for rehabilitation services. SQL and MongoDB are used for data management, while Python libraries such as Pandas, NumPy, and Scikit-learn are used for data processing and predictive modeling.

Technologies Used
Python – Data preprocessing, analysis, and machine learning model development.
Pandas – Data cleaning, transformation, and manipulation.
NumPy – Numerical computations and array operations.
Scikit-learn – Machine learning model training, evaluation, and feature engineering.
Matplotlib & Seaborn – Data visualization and exploratory data analysis (EDA).
Jupyter Notebook – Interactive development, experimentation, and documentation.
Git & GitHub – Version control and project collaboration.
